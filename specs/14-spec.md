# 用户登录模块

## 需求概述

## 需求描述

实现一个用户登录页面，支持邮箱+密码登录。

## 功能点

1. 登录表单（邮箱、密码输入框）
2. 表单前端校验
3. 调用后端登录接口
4. 登录成功后跳转首页
5. 登录失败显示错误提示

## 技术栈

- 前端：HTML + CSS + JavaScript
- 无需后端（Mock 数据）
